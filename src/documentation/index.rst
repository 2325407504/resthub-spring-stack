===============================
RESThub 1.1 Documentation
===============================

RESThub is an Open Source integration framework, based on `Spring 3 <http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/>`_ and `Jersey <http://jersey.java.net/>`_, designed to allow you to build applications following KISS, DRY and REST principles.
